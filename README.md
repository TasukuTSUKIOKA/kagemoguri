# 影もぐり / Kagemoguri

光を避けて、影へもぐるブラウザゲーム。  
見つかりそうになったら煙幕で立て直しながら、生き残った時間を競う。

## Play

GitHub Pages で公開予定

## Controls

- Move: `←` `→` / `A` `D` / on-screen buttons
- Smoke: `Shift` / smoke button
- Start / Pause / Resume / Retry: `Space` / start button
